Этот скрипт рекурсивно обходит страницы выбранного сайта, собирает метрику времени загрузки каждой страницы и составляет топ страниц по времени их открытия.

Особенности:
- Ограничение по домену - краулер не уходит на внешние сайты.
- Строгая глубина обхода (max_depth).
- Лимит страниц (max_pages) для предотвращения бесконечного сканирования.
- Измерение времени загрузки страницы через `window.performance.timing`.
- Обработка относительных ссылок и поддоменов.

При активации скрипта вводятся данные:
- URL-адрес сайта
- Глубина обхода сайта
- Лимит страниц
